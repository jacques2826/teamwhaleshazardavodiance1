Login For New Drivers

There are 3 steps that a new driver must know to be able to access the system.

1. The Driver must have made previous contact with (via email) an administrative user so that they can be added to the database, and given a temporary user name and password in so that they can gain access to the system.

2. Once logged the driver will be prompted to create a new password- creating a password that is between 8-15 characters long with 1 upper case, 1 lower case and 1 special character.

3. The driver will then be able to view their route based on their driver id, they will also be able to see the associated manager they report to as well as their home lot.
